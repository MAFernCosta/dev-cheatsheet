dev-cheatsheet

Linux Commands Cheat Sheet

Quick reference extracted and organized into categories. Copy, tweak, and extend as needed.

File Commands

Command Description
ls List the content of the current directory
ls -R List subdirectories recursively
ls -a List all content, including hidden files
ls -l List content with detailed information
pwd Show the current directory
cd folder1 Change directory to folder1
cd Return to $HOME directory
cd .. Go up one directory
cd - Return to the previous directory
cp source destination Copy file
cp -r source destination Copy folder recursively
mv source destination Move or rename file
rm file1 Remove file
rm -f file1 Force remove file without prompt
rm -r folder Remove folder and its contents recursively
mkdir foldername Create a new folder
rmdir foldername Remove an empty folder
file file1 Show file type
cat file1 file2 Concatenate files
cat > file1 Write standard input to file
less file1 View file (paged)
head file1 Show first 10 lines
tail file1 Show last 10 lines
chmod 777 file Change file permission
chown user:group file Change file ownership
ln -s source destination Create symbolic link

System Commands

Command Description
uname -a Show system and kernel info
head -n1 /etc/issue Show distribution information
mount Show mounted filesystems
date Show system date
uptime Show system uptime
whoami Show current username
w Show who is online
man command Show manual page
mount -o loop cdrom.iso /mnt/dir Mount ISO image
cat /proc/partitions Show registered partitions
grep MemTotal /proc/meminfo Show total RAM
grep "model name" /proc/cpuinfo Show CPU info
lspci -tv Show PCI devices
lsusb -tv Show USB devices
!! Repeat last command
exit Logout / exit shell

File Searching

Command Description
grep pattern files Search for pattern in files
grep -i Case-insensitive search
grep -r Recursive search
grep -v Invert match
grep -o Show only matched part
locate file1 Find by name (database)
whereis command Locate binary/source/manual
which app Show path of command
look string file1 Lines starting with string
find /dir/ -user name Find files owned by user
find /dir/ -mmin num Modified less than num minutes ago
find /dir/ -name 'name*' Names starting with value

Encryption & Compression

Command Description
gpg -c file Encrypt file (symmetric)
gpg file.gpg Decrypt file
tar -cf archive.tar foo bar Create tar archive
tar -xf archive.tar Extract tar archive
tar -czf archive.tar.gz foo bar Create gzip-compressed tar
tar -xzf archive.tar.gz Extract gzip-compressed tar
tar -cjf archive.tar.bz2 foo bar Create bzip2-compressed tar
tar -xjf archive.tar.bz2 Extract bzip2-compressed tar
gzip file1 Compress file (to .gz)
gzip -d file1.gz Decompress .gz file

Process Management

Command Description
ps Snapshot of processes
top Interactive process viewer
kill pid Kill process by PID
pkill name Kill by process name
killall name Kill all with matching name

Keyboard Shortcuts

Shortcut Action
Ctrl + Shift + C Copy highlighted text (terminal)
Ctrl + Shift + V Paste copied text (terminal)
Ctrl + Z Suspend current process
Ctrl + C Interrupt/stop current command
Ctrl + R Search command history
Ctrl + U Delete from start of line
Ctrl + K Delete to end of line
Ctrl + A Move to start of line
Ctrl + E Move to end of line

SSH & SCP

Command Description
ssh $USER@$HOST Connect to remote host
ssh $USER@$HOST command Run remote command
ssh $USER@$HOST -p 1234 Connect on specific port
scp file1 $USER@$HOST:file1 Copy file to remote host
scp $USER@$HOST:file1 file1 Copy file from remote host
scp -r foo $USER@$HOST:/bar Copy directory to remote host

Disk & Usage

Command Description
df -h Show free space (human readable)
df -i Show free inodes
du -h folder Usage of each item in folder
du -sh folder Total size of folder

Packages & System Info

Command Description
dpkg -i package.deb Install Debian package
rpm -Uvh package.rpm Install RPM package
fdisk -l List disks/partitions
free Show memory and swap usage
Tip: not all distributions behave identically; double-check options if something looks off.
